- **About** — Summary, key stats (3+ years, 15+ projects)
- **Services** — Agentic AI, Full-Stack Platforms, AI Automation
- **Experience** — Freelance AI Engineer (2023–Present)

- **Skills** — Tech icon grid (29 technologies across AI, Backend, Frontend, DevOps)
